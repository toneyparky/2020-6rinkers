# 3. :computer: 핵심 기능
* 칵테일 정보를 제공합니다.
    * 이름, 이미지, 어원, 특징, 도수, 맛, 레시피가 있습니다.
* 칵테일을 검색할 수 있습니다.
    * 칵테일 이름과 태그를 이용합니다.
* 칵테일을 추천합니다.
    * 사용자의 취향을 기반으로 합니다.
* 칵테일을 즐겨찾기할 수 있습니다.
